Chaque élément inscrit dans les parties suivantes sont des contraintes pour la réalisation du projet et doivent être respectées.

Technologies

Selon l'environnement, vous utiliserez des technologies différentes : PowerShell pour Windows et Python ou Shell pour Linux.

Analyse des fichiers

Vous utiliserez des outils d'analyse antivirus/malware.

Vous intégrerez des scripts personnalisés pour détecter les types de fichiers et effectuer des validations spécifiques.

Automatisation

Vous devrez prévoir un service de surveillance de fichiers afin de déclencher les actions à chaque modification des partages.

Versioning et gestion des sources

Tous vos scripts réalisés doivent être versionnés sur GitLab ou GitHub pour simplifier la collaboration, le suivi des modifications et le partage.
